# Run safety detection on-device only

Cereveil will run Scam Text Detection and NSFW Screen Detection on the Child Device rather than sending raw screen content to the backend. Child Mode may create a local Safety Intervention and send a metadata-only Safety Alert to Convex, but raw accessibility text, OCR text, screenshots, screen recordings, audio, and captured pixels are never uploaded or retained by Guardian-facing systems. This may constrain model choice and make server-side reprocessing unavailable, but keeps sensitive content involving minors out of backend storage and preserves the privacy boundary promised by on-device inference.
