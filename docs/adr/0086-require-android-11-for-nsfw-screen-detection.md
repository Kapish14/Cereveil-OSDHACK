# Require Android 11 for NSFW Screen Detection

Cereveil will remain installable on Android 8 and newer. Scam Text Detection may be configured on every supported Child Device, while NSFW Screen Detection requires Android 11 or newer because it depends on the accessibility screenshot API introduced in Android 11. Guardian Mode shows only the NSFW section as unavailable for an older Child Device; the independent schema v3 detector sections allow scam-only protection without loading or partially pretending to apply the NSFW detector.
