# Separate Guardian cache from Child operational state

Cereveil will use Room for local persistence but treat Guardian Mode and Child Mode data differently. Guardian Mode local data is primarily a cache of Household state, Child Profiles, policies, Guardian Notices, summaries, and dashboard data that can show stale/offline status while reconnecting. Child Mode local data is operational state required for enforcement, including the Active Enrollment reference, last accepted Supervision Policy, applied policy version, offline upload queue, Supervision Health state, current Access Grants, and short-lived local incident suppression fingerprints. This adds separate persistence boundaries, but keeps offline Child enforcement reliable without treating Guardian cache data as authoritative.
