# Upload a minimal latest-only App Catalog

Child Mode will upload the latest known App Catalog so Guardian Mode can select installed apps for App Blocks and other app-specific supervision features. The catalog contains only user-launchable package names, device-resolved display labels, installation presence, and catalog synchronization time; it excludes icons, permissions, APK metadata, installation timestamps, system components, and catalog history, and removed entries are deleted rather than retained as uninstall history. Protection Setup must explicitly disclose this latest installed-app sharing to the Child and Guardian. This accepts collecting a bounded view of installed apps while avoiding `QUERY_ALL_PACKAGES`-style broad inventory and long-term installation profiling.
