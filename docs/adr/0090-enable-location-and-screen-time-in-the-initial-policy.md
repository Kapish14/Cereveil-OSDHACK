# Enable Location Sharing and Screen Time in the Initial Policy

Cereveil will enable Location Sharing and Screen Time in version 1 of every newly created Child Profile's Supervision Policy. Guardian Mode must disclose these defaults before the Guardian creates the profile and explain that either feature can be turned off in Settings. Collection still cannot begin before a Child Device is enrolled, has the required Android capabilities, and applies the policy. Existing Child Profiles keep their current policy values; this is not a migration. This supersedes ADR-0081 only where it requires Location Sharing to be off initially, clarifies the privacy-default language in ADR-0052, and satisfies ADR-0082's initial Screen Time disclosure requirement through Child Profile creation.
