# End Supervision by revoking device access and deleting child data

Cereveil will process End Supervision as a Guardian-approved backend workflow that revokes Child Device access and hard-deletes child-specific supervision data. Convex will verify Guardian authority over the Child Profile, revoke Child Device Credentials, revoke or end the Active Enrollment, invalidate Child Device FCM tokens, delete pending commands, grants, alerts, summaries, location state, Remote Audio state, Supervision Health, policy application state, policies, the Child Device record, and the Child Profile. If the Child Device is online, it receives an end-supervision command and clears local operational state; if offline, backend revocation still takes effect and old Child Device credentials no longer authorize sync. This keeps End Supervision distinct from Replace Child Device, which preserves the Child Profile and retained data for a new enrollment.
